# DevTools and General Tips

## Easy & Free ChatGPT Integration
 - `link/?prompt=${user_prompt}`
 - route to this url with user prompt.

## Context Mapping for LLMs
 - Integrate structure tree in a project to save tokens as it provides fast & easy context of the codebase to the model making it easy to understand and get result.

## Laravel Overview
 - Free PHP framework (faster & easier) for web dev. Many Built in features like auth, routing, ORM, DB migration etc.
 - Follows MVC architecture.
