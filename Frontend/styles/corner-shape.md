# corner-shape
- New CSS property that lets you make custom shapes.
- it is a shorthand property for corner-shape-top-left, corner-shape-top-right, corner-shape-bottom-left, corner-shape-bottom-right

## check 
- css tricks(https://css-tricks.com/what-can-we-actually-do-with-corner-shape/) or MDN for docs

## example 
```css
.box {
  corner-shape: bevel/scoop/square/round/...;
  corner-shape: superellipse(...custom values...);
}
```
